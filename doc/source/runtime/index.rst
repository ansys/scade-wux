Runtime interfaces
==================

This section is the reference for the C/C++ interfaces of the code
generated by the generation services of this package, as well as their
runtime code.

* The header files are in the ``site-packages/ansys/scade/wux/include`` directory.
* The runtime implementation files that are not generated are in the
  ``site-packages/ansys/scade/wux/lib`` directory.
